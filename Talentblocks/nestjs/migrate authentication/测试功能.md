## 测试顺序总结

|步骤|请求|预期|
|---|---|---|
|1|`POST /auth/login`|200 + tokens|
|2|`GET /auth/me` + Bearer token|200 + user info|
|3|`GET /auth/me` 不带 token|401|
|4|`POST /auth/refresh` + refreshToken|200 + 新 tokens|
|5|`POST /auth/refresh` + 旧 refreshToken|401（重用检测）|
|6|`POST /auth/logout` + 新 refreshToken|200|
|7|`POST /auth/refresh` + 已 logout 的 token|401|

按这个顺序走一遍，所有功能就验证完了。遇到任何问题告诉我。
