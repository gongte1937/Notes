# Notes

这是一个使用 [Obsidian](https://obsidian.md/) 工具管理的个人笔记仓库。

## 简介

本仓库用于记录和整理日常学习、工作中的各类知识笔记，通过 Obsidian 进行可视化管理和双向链接。

## 关于 Obsidian

Obsidian 是一款强大的知识管理工具，被称为"你的第二大脑"。它采用本地优先的设计理念，所有数据都存储在本地，完全由你掌控。

### 核心特点

- 📝 **基于 Markdown** - 纯文本格式，永不过时，可随时迁移
- 🔗 **双向链接** - 像维基百科一样连接笔记，构建知识网络
- 📊 **知识图谱** - 可视化展示笔记间的关联关系
- 🔌 **插件生态** - 丰富的社区插件，扩展无限可能
- 💾 **本地存储** - 数据完全由自己掌控，支持任何同步方案
- ⚡ **快速搜索** - 全库搜索，快速定位内容
- 🎨 **高度定制** - 主题、CSS 片段，打造专属笔记环境
- 📱 **跨平台** - 支持 Windows、macOS、Linux、iOS、Android

### 核心功能详解

#### 1. 双向链接 (Backlinks)

在任何笔记中使用 `[[笔记名称]]` 即可创建链接：
- `[[JavaScript]]` - 链接到 JavaScript 笔记
- `[[JavaScript#函数]]` - 链接到特定标题
- `[[JavaScript|JS]]` - 自定义显示文本

被链接的笔记会自动显示反向链接，帮助你发现笔记间的关联。

#### 2. 知识图谱 (Graph View)

- **全局图谱**：展示所有笔记的关联关系
- **局部图谱**：聚焦于当前笔记的相关连接
- **可交互**：点击节点可直接打开笔记
- **可筛选**：按标签、文件夹等过滤显示

#### 3. 标签系统

使用 `#标签` 组织笔记：
- `#编程` - 一级标签
- `#编程/JavaScript` - 嵌套标签
- 支持标签面板，快速浏览所有标签

#### 4. 搜索功能

- **全文搜索**：快速查找所有笔记中的内容
- **正则表达式**：支持高级搜索模式
- **搜索运算符**：如 `file:`, `tag:`, `path:` 等

#### 5. 模板功能

创建笔记模板，提高效率：
- 日记模板
- 会议记录模板
- 读书笔记模板

#### 6. 工作区布局

- 多窗格编辑
- 分屏预览
- 侧边栏管理
- 自定义面板布局
