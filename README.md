# Notes

这是一个使用 [Obsidian](https://obsidian.md/) 工具管理的个人笔记仓库。

## 简介

本仓库用于记录和整理日常学习、工作中的各类知识笔记，通过 Obsidian 进行可视化管理和双向链接。

## 关于 Obsidian

Obsidian 是一款强大的知识管理工具，具有以下特点：

- 📝 基于 Markdown 格式的笔记编辑
- 🔗 支持笔记间的双向链接
- 📊 可视化的知识图谱
- 🔌 丰富的插件生态系统
- 💾 本地存储，数据完全由自己掌控

## 笔记内容

目前包含的笔记主题：

- **Markdown.md** - Markdown 语法参考指南

## 使用方法

1. 安装 [Obsidian](https://obsidian.md/)
2. 在 Obsidian 中打开此目录作为 vault
3. 开始浏览和编辑笔记

## 版本控制

本仓库使用 Git 进行版本控制，方便追踪笔记的变更历史和跨设备同步。
