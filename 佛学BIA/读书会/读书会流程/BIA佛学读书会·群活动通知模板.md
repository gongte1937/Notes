# BIA佛学读书会·群活动通知模板

> 本模板根据 [[BIA佛学读书会·会议操作手册]] 和 [[BIA佛学读书讨论会活动策划方案]] 整理，适用于每期读书会在群内发布活动通知、会前提醒和当天提醒。

## 一、通知必备信息

每次群内通知建议至少包含以下内容：

- 活动主题
- 阅读书目与章节范围
- 日期与时间
- 主讲人 / 主持人
- 活动形式（线上 / 线下）
- 地点 / Zoom 链接
- 预计总时长
- 预计阅读时长
- 本次讨论要点
- 2 至 3 个引导问题
- 报名方式
- 适合对象
- 参与者会前准备

## 七、建议写法

- 语气尽量温和、开放、初学者友好。
- 不要把通知写成课程宣传，重点是共读与讨论。
- 讨论问题尽量按“进入文本 → 理解概念 → 联系生活”推进。
- 每次通知都尽量写清楚阅读范围，不要只写书名。
- 若为线下活动，可补一句是否提供茶点或是否需要提前到场。
- 若为线上活动，可补一句请提前 5 分钟进入会议室。
